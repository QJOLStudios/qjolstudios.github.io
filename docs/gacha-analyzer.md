## 使用说明

1. 在游戏中点击 **"导出记录"** 按钮
2. 点击 **"复制到剪贴板"** 复制 JSON 数据
3. 回到此页面，将 JSON 粘贴到输入框中
4. 即可查看详细的抽卡分析统计！
